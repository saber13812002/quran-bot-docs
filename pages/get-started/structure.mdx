# ساختار پروژه

## معرفی
این بخش به توضیح ساختار کلی پروژه می‌پردازد. درک صحیح این ساختار به شما کمک می‌کند تا بهتر بتوانید با پروژه کار کنید.

## پوشه‌های اصلی
- `src/`: کدهای اصلی پروژه
- `components/`: کامپوننت‌های قابل استفاده مجدد
- `pages/`: صفحات پروژه
- `styles/`: فایل‌های استایل
- `public/`: فایل‌های استاتیک

## توضیحات تکمیلی
هر پوشه وظیفه مشخصی در پروژه دارد و باید فایل‌های مرتبط در پوشه مناسب خود قرار بگیرند.
